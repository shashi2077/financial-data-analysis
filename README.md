# Financial Data Analysis Project

**Author:** Shashikant Yadav  
**Tech Used:** Python, Pandas, Matplotlib, Excel (openpyxl)

---

## 📌 Project Overview

This project analyzes financial datasets to identify trends, performance indicators, and business insights. It demonstrates end-to-end data analysis — from raw data cleaning to visual reporting.

---

## 📊 What This Project Does

- **Data Loading & Cleaning** — Loads CSV data, checks for missing values and duplicates
- **KPI Calculation** — Revenue, Expenses, Profit, Profit Margin, MoM Growth
- **Trend Analysis** — Month-over-month performance tracking
- **Visualizations** — 4 charts: Bar chart, Line trend, Margin %, Sales vs Customers
- **Excel Report** — Auto-generates a multi-sheet Excel report with KPI Summary & Trend Analysis

## ▶️ How to Run

**1. Install dependencies:**
```bash
pip install pandas matplotlib openpyxl
```

**2. Run the script:**
```bash
python financial_analysis.py
```

**3. Outputs generated:**
- `Financial_Analysis_Report.xlsx` — Excel report with 3 sheets
- `charts/financial_analysis_charts.png` — 4 visualizations
